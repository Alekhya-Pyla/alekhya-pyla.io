Hey!

My experience and areas of interests lie in Backend Software Development - Infrastructure and Databases, Machine Learning - NLP and RL. Research

Learning to Query Internet Text for Informing Reinforcement Learning Agents

Worked on improving the generalization capabilities of reinforcement learning agents using Natural Language Processing. Experimented querying on T5, Unified QA, and RNN models using unstructured text on the internet, published in RLDM 2022.

Incipient Fault Diagnosis in Induction motors using Deep Learning

Devised a novel fault diagnosis methodology in induction motors using Deep learning techniques of 1D-CNN and LSTM, published at IEEE IAS Annual Meeting, 2018.

Academic Projects

Stock Market Trend Analysis using Stream Processing

Designed a middleware architecture which performs analysis on variable frequency streams of stocks, news and tweets using Flink and Kafka along with load-shedding.

Hate Speech Detection using NLP

Devised Hate Speech Detection mechanism using the transfer learning approach in NLP, using DistilBERT model in Pytorch and explored different machine learning techniques such as Random Forest, SVM for classification.

Longformer and Summarization evaluation

Evaluated the efficiency of Longformer model for long document summarization task using Longformer Encoder Decoder model and performed comparison using metrics of ROUGE, BLEU.

Search Engine for UCI ICS website

Built a search engine for UCI website by developing an inverted index of all the web-pages in UCI domain and implementing ranking algorithm using techniques of Cosine similarity, Page-Rank in Python.

Hierarchical Modeling for Image Compression

Developed efficient image compression technique using Variational Auto-Encoder model and compared with traditional JPEG compression.

Exploring DIfferential eVasive Attack attack on Edge Models

Implemented evasive attack by developing input perturbations to add for quantized edge models and original models of MobileNet, ResNet and DenseNet.

Level Generation for Angry Birds with Sequential VAE

Built automatic levels in Angry Birds game using Deep Generative Modelling.
